# Отчёт проверки MVP

Проверено 20 июня 2026 года:

- синтаксис JavaScript;
- отсутствие дублирующихся HTML ID;
- соответствие ID, используемых JavaScript, элементам интерфейса;
- запуск интерфейса;
- мобильная ширина 412 px;
- открытие формы заявки;
- заполнение клиента, телефона, адреса и района;
- добавление размеров и материала ковра;
- выбор загрязнения;
- расчёт площади;
- формирование заголовка и описания события;
- проверка маршрута района;
- сохранение в локальном демо-режиме;
- отсутствие ошибок JavaScript в тестовом сценарии.

Реальная запись в Google Calendar не проверялась, потому что для неё требуется OAuth Client ID владельца и разрешённый HTTPS-домен приложения.
