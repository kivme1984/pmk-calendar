# ПМК Календарь — MVP

Лёгкая PWA-оболочка Google Calendar для оформления заявок «Промой Ковер» в едином формате.

## Уже работает

- экраны «Сегодня» и «Неделя»;
- форма новой заявки и редактирование;
- несколько ковров в одной заявке;
- загрязнения, допуслуги, стоимость и скидка;
- подсказки по маршрутам районов;
- предупреждение о пересечении времени;
- предупреждение о минимальном заказе;
- кнопки звонка и построения маршрута;
- демо-режим без Google: данные хранятся в localStorage;
- создание и обновление событий Google Calendar через Calendar API;
- скрытое структурированное хранение данных заявки в extendedProperties события;
- установка на Android как PWA.

## Быстрый запуск без Google

Приложение нельзя полноценно запускать двойным нажатием по `index.html`, потому что PWA и OAuth требуют HTTP(S).

В папке проекта выполните:

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.

## Подключение Google Calendar

1. Создайте проект в Google Cloud Console.
2. Включите **Google Calendar API**.
3. Настройте OAuth consent screen.
4. Создайте OAuth Client ID типа **Web application**.
5. Добавьте адрес приложения в **Authorized JavaScript origins**, например:
   - `http://localhost:8080` для теста;
   - адрес Cloudflare Pages / GitHub Pages для рабочей версии.
6. Откройте «Настройки» приложения и вставьте Client ID.
7. Нажмите «Подключить Google».

Используемый scope: `https://www.googleapis.com/auth/calendar.events`.

## Публикация

Папку можно разместить как статический сайт на Cloudflare Pages, GitHub Pages, Netlify или другом HTTPS-хостинге. Сборка не требуется.

## Важно

В текущем MVP Google Calendar является главным хранилищем рабочего расписания. Локальные черновики сохраняются только на устройстве. Архитектура формы и формат данных подготовлены для будущего подключения собственной базы и перехода к CRM.
