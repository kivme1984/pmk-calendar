# Задание для Codex: опубликовать «ПМК Календарь»

Нужно опубликовать статическое PWA-приложение из этой папки без изменения бизнес-логики.

## Требования

1. Создай отдельный GitHub-репозиторий `pmk-calendar` или используй указанный владельцем репозиторий.
2. Загрузи содержимое папки в корень репозитория.
3. Разверни приложение через Cloudflare Pages.
4. Build command не нужен.
5. Output directory — корень проекта (`/`).
6. Убедись, что по HTTPS доступны:
   - `/index.html`
   - `/app.js`
   - `/styles.css`
   - `/manifest.webmanifest`
   - `/sw.js`
   - `/icons/icon-192.png`
   - `/icons/icon-512.png`
7. Проверь установку PWA на Android и работу демо-режима.
8. Не добавляй Google Client Secret в код или репозиторий.
9. После получения OAuth Client ID пользователь вставит его через экран «Настройки» приложения.
10. Сообщи владельцу итоговый HTTPS-адрес, который нужно добавить в Google Cloud в `Authorized JavaScript origins`.

## Проверка после публикации

- создать тестовую заявку;
- проверить подсказку маршрута;
- проверить предупреждение о пересечении;
- обновить страницу и убедиться, что локальная заявка сохранилась;
- установить приложение на Android через меню браузера «Добавить на главный экран».
